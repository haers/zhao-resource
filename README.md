# 小赵资源站

小赵资源站整理夸克网盘资源与常用学习入口，覆盖学生教育、职业考试、编程与 AI、设计剪辑、电子书、健康生活、软件工具、音乐音频等主题。主站按分类浏览，GitHub 仓库保留站点源码和本地运行方式。

## 本地预览与部署

需要 Node.js LTS 和 pnpm 10：

```bash
pnpm install
pnpm docs:dev
pnpm docs:build
```
